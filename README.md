
## Getting Started

```bash
docker-compose -f docker-compose.dev.yaml up --build -d
```

```bash
docker-compose -f docker-compose.dev.yaml down -v
```

Swagger: http://localhost:5000/swagger/index.html

Grafana: http://localhost:3000/?orgId=1
